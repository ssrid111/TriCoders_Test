# 🤝 Contributing Guide

Welcome! 🎉
This repository is designed to help you complete a simple open-source contribution with guidance from an AI agent.

---

## 🧩 Your Task

👉 Add your GitHub username to:

```
tasks/hello.txt
```

---

## 🛠 Step-by-Step Instructions

### 1. Fork the repository

Click the **Fork** button on the top right of this repository.

---

### 2. Clone your fork

```
git clone https://github.com/YOUR-USERNAME/agent-contribution-test.git
cd agent-contribution-test
```

---

### 3. Create a new branch

```
git checkout -b add-your-name
```

---

### 4. Edit the required file

Open:

```
tasks/hello.txt
```

Add your GitHub username under the list:

```
- your-username
```

---

### 5. Commit your changes

```
git add .
git commit -m "Added my username"
```

---

### 6. Push your branch

```
git push origin add-your-name
```

---

### 7. Create a Pull Request

* Go to your fork on GitHub
* Click **Compare & pull request**
* Submit your PR

---

## 🤖 AI Agent Support

This repository is used to demonstrate an AI agent that:

* Tracks your progress
* Detects if you get stuck
* Provides real-time guidance

If you are unsure what to do next, follow the steps above or use the AI assistant on the website.
