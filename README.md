# 🚀 Agent Contribution Test Repo

This repository is designed to test an AI agent that helps users complete their first open-source contribution.

## 🧩 Task

Your goal is simple:

👉 Add your GitHub username to `tasks/hello.txt`

---

## 🛠 Steps

1. Fork this repository
2. Clone your fork
3. Open `tasks/hello.txt`
4. Add your username
5. Commit and push
6. Create a Pull Request

---

## 🤖 Why this repo?

This repo is used to demonstrate an AI agent that:
- Tracks your progress
- Detects if you get stuck
- Guides you step-by-step

---

Good luck and happy contributing! 🎉
